My Experience Creating a Portfolio Website

A story about my experience when making a portfolio website using HTML, CSS, and Tailwind CSS.

Orientation

Last month, I had an interesting but difficult experience while creating my portfolio website for a school project. I wanted to make a modern website with a purple theme using HTML, CSS, and Bootstrap. At first, I felt excited because I thought the process would be easy and fun.

Events

When I started coding the website, many problems appeared. The navbar could not work properly, some images were too big, and the buttons did not respond when I clicked them. I felt confused because the code looked correct to me.

After that, I tried to fix the errors by checking the code carefully. Sometimes I accidentally deleted important parts of the code, which made the website worse. I spent a lot of time searching for solutions on the internet and watching tutorial videos. I also practiced many times to understand how Bootstrap worked.

Even though I was tired and frustrated, I kept trying to improve my website little by little. Finally, after many corrections, the website looked better and all the features worked properly.

Reorientation

From this experience, I learned that making a website is not as easy as I imagined. It requires patience, hard work, and carefulness. However, I felt very proud because I did not give up and successfully finished my portfolio website.
